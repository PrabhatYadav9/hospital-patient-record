# Hospital Patient Records Management System

A simple Python console-based Hospital Patient Records Management System that allows users to manage patient records.

## Features

- Add Patient
- View All Patients
- Search Patient by ID or Name
- Update Patient Details
- Discharge Patient
- Delete Patient
- Auto-generated Patient ID

## Technologies Used

- Python 3

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/hospital-patient-records.git
   ```

2. Open the project folder:
   ```bash
   cd hospital-patient-records
   ```

3. Run the program:
   ```bash
   python main.py
   ```
